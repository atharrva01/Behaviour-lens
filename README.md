# BehaviorLens

Real-time user behavior observability system built with Go.

## Overview

BehaviorLens is a lightweight observability platform designed to track and analyze user behavior in real-time. Built for developers who need actionable insights into how users interact with their applications.

## Status

🚧 **Under Active Development** - This project is currently in early stages.

## Tech Stack

- **Backend**: Go
- **Frontend**: React (planned)
